Программа для расшифровки паролей, хранящихся в файлах 1CV8Clst.lst или 1CV8Reg.lst на кластере 1С Предприятие.

# requirements
pip3 uninstall crypto
pip3 install pycryptodome


# usage 
python 1c_pass_cracker.py <имя_файла>
